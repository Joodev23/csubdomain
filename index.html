Create Subdomain

Letters, numbers & hyphens only

API Documentation

POST /api/create-subdomain

Creates a new subdomain pointing to the given IP.

Headers

Authorization: Bearer JooooModdssAlicia11112025
Content-Type: application/json

Body (JSON)

{
  "subdomain": "panelku",
  "ip": "192.168.1.1",
  "domain": "vortexx.web.id"
}
Example cURL 
curl -X POST https://yourdomain/api/create-subdomain \
  -H "Authorization: Bearer JooooModdssAlicia11112025" \
  -H "Content-Type: application/json" \
  -d '{
        "subdomain": "panelku",
        "ip": "192.168.1.1",
        "domain": "vortexx.web.id"
      }'

200 OK

{
  "success": true,
  "data": {
    "subdomain": "panelku.vortexx.web.id",
    "ip": "192.168.1.1"
  }
}

4xx / 5xx

{
  "success": false,
  "error": "Subdomain already exists"
}
